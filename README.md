# To-Do List

A simple and interactive to-do list built using HTML, CSS, and JavaScript.  
Allows users to add, mark, and remove tasks easily.

## Tech Used
- HTML
- CSS
- JavaScript

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Responsive design
